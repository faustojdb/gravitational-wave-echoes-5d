## 🚀 **Quick Start**

### **Reproduce Main Results**
```bash
git clone https://github.com/faustojdb/gravitational-wave-echoes-5d.git
cd gravitational-wave-echoes-5d

# Install dependencies
pip install -r requirements.txt

# Run main analysis
python analysis/ligo_data_analysis.py

# Generate statistical summary
python analysis/statistical_analysis.py

# Create publication figures
python analysis/generate_figures.py
```

### **Verify Echo Detection**
```bash
# Test echo detection algorithm
python analysis/echo_detection.py --event GW150914

# Run full GWTC-1 analysis
python analysis/systematic_analysis.py --catalog GWTC1

# Compare with theoretical predictions
python analysis/klein_simulation.py --verify-predictions
```
